# Sistema DN - Gestão de Chamados e Documentos Normativos (PWA)

O **Sistema DN** é uma solução Web Progressiva (PWA) desenvolvida para otimizar o fluxo de trabalho dos técnicos em infraestrutura que atuam em campo.
O objetivo principal do sistema é permitir que técnicos acessem chamados de suporte, gerenciem movimentações de equipamentos e preencham Documentos Normativos (DNs) de forma integrada,
consumindo diretamente as APIs da plataforma Movidesk, eliminando a necessidade de processos manuais e burocráticos.

## Objetivo
Transformar a operação da infraestrutura, permitindo que técnicos atendam chamados, registrem movimentações e coletem assinaturas digitais diretamente via dispositivos móveis (celulares ou tablets),
garantindo a baixa automática nos chamados via API.

## Funcionalidades Principais
O sistema centraliza 6 tipos de Documentos Normativos essenciais para a operação:

1.  **DN-25:** Relatório de Visita Técnica.
2.  **DN-75:** Entrega de Equipamentos.
3.  **DN-75:** Recolha de Equipamentos.
4.  **DN-75:** Retirada de Equipamentos.
5.  **DN-75b:** Substituição de Equipamentos.
6.  **DN-75c:** Movimentação de Equipamentos.

### Destaques da Solução
*   **Integração API:** Consulta e baixa de chamados diretamente do Movidesk.
*   **Assinatura Digital:** Módulo nativo para coleta de assinatura via *touchscreen*.
*   **Design Responsivo:** Interface otimizada para Desktop, Tablets e Celulares.
*   **PWA:** Funcionalidade de aplicação web progressiva para instalação em dispositivos.

## Stack Tecnológica
*   **Backend:** Java com framework Spring Boot.
*   **Frontend:** HTML5, CSS3, JavaScript (Responsivo).
*   **Banco de Dados:** MySQL.
*   **Utilitários:** Lombok (redução de boilerplate).
*   **IDE:** IntelliJ IDEA.
*   **Deploy:** Vercel.

## Status do Projeto
O sistema encontra-se em **fase de desenvolvimento ativo e contínuo**, passando por atualizações frequentes para integrar novas funcionalidades e refinar o fluxo de trabalho dos documentos normativos (DNs).
